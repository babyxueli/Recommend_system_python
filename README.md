# 开发文档

interface.py 里面定义了接口函数，前端所有需要的以及要更改的数据请从这里开始,需要另外的功能请自行在这里添加，并详细说明。

system_object.py 里定了数据结构的组成形式

src 文件夹为存储用户头像，电影画报的图片等

spider 存放爬虫

web文件将全部放在web_recommend文件夹中。
